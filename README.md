# SNES_12
SNES example code

2020 Doug Fraker

Learn to program a Super Nintendo game from scratch.
This program shows color math examples.
Press A,B,X, or Y to toggle through them.

0-no effect
1-add
2-add and half
3-subtract
4-subtract and half
5-fixed color, add 
6-using the "always clip" main, and add mode, 
  shows only the sub screen
